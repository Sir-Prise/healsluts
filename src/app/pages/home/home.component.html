Playing the healer in multiplayer online games is fun on it's own, but you can make it even more fun by connecting a vibrator! The better you play, the higher is the vibration.

This website uses a browser feature which is usually used for screen sharing, but here it's used to check the color of certain pixels on the screen to determine how you play. All analysis happens in your browser, your game screen is not actually shared.

For now, this website works for Overwatch and analyzes the "on-fire meter" below the health bar. When you're dead the vibration should be zero, otherwise the intensity should match the value of the meter.
